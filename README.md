# AutoBazar Telegram Mini App — MVP

## Уже працює
- 5 територій по 20 паркомісць;
- розподіл машин за ціною;
- створення оголошення;
- персонаж ходить кнопками, стрілками та WASD;
- перегляд детальної інформації;
- повідомлення продавцю;
- статус продавця онлайн/офлайн;
- оголошення залишаються після виходу;
- Telegram-авторизація з перевіркою на сервері;
- демонстраційний режим у браузері.

## Запуск
Потрібен Node.js 20+.

```bash
npm install
```

Створи `.env`:

```env
PORT=3000
BOT_TOKEN=
DEMO_MODE=true
```

Потім:

```bash
npm start
```

Відкрий `http://localhost:3000`.

## Для Telegram
Додай токен бота:

```env
BOT_TOKEN=ВАШ_ТОКЕН
DEMO_MODE=false
```

Розмісти весь проєкт на сервері з HTTPS, наприклад Render або Railway.
GitHub Pages не підходить для всього проєкту, бо тут є Node.js і Socket.IO.

## П’ята територія
Від $20 000.

## Обмеження MVP
- один продавець — одна активна машина;
- дані зберігаються у JSON-файлі;
- немає фото;
- повідомлення видно всередині гри, але Telegram-сповіщення ботом ще не додані;
- для великої кількості користувачів потрібна база PostgreSQL.
